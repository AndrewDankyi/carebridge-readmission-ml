ML Pipeline Architecture
1
Data Ingestion & ETL
SQL schema loads synthetic EHR records. PostgreSQL extracts patient demographics, admission history, diagnoses, and medication data into structured tables.
PostgreSQLschema.sql
2
Feature Engineering
28 raw features → 42 engineered features. ICD-9 grouping, age binning, discharge disposition encoding, medication change flags, and comorbidity indexing.
pandasNumPy
3
Class Balancing (SMOTE)
Target class (readmitted) is ~11% of records. SMOTE applied on training set only to prevent data leakage. Balanced ratio: 1:1 for training.
imbalanced-learn
4
Model Training & Tuning
GridSearchCV over 4 classifiers (RF, XGB, LR, DT). 5-fold stratified cross-validation. Best params: n_estimators=200, max_depth=12, min_samples_split=5.
scikit-learnXGBoost

Explainability (SHAP)
TreeExplainer generates per-patient SHAP values. Top drivers: prior admissions, LOS, diagnosis count. Supports clinical interpretability for discharge planning.
